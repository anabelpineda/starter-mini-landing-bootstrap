# starter-mini-landing-bootstrap

PRÁCTICA 1 — Mini Landing Profesional con Bootstrap

📌 Práctica previa al examen práctico
👤 Trabajo individual
⏱ Tiempo estimado: 1 clase

🎯 Objetivo
Construir desde cero una mini landing page profesional utilizando Bootstrap (CDN).

El tema es libre.
Puede ser sobre:

⚽ Fútbol
🎮 Videojuegos
🎬 Cine
🎵 Música
💻 Tecnología
🏋️ Deportes
🏥 Salud
📱 Apps
🚀 Espacio
O cualquier otro tema que te interese

Lo importante no es el tema.
Lo importante es la estructura y el diseño profesional.

📂 Instrucciones Generales

Crear un archivo llamado: index.html
Usar Bootstrap vía CDN.
No usar plantillas descargadas.
No reutilizar código de tu proyecto anterior.
El diseño debe ser limpio y coherente.

🧩 Estructura obligatoria
1️⃣ Navbar Profesional
Debe incluir:
navbar
navbar-expand-lg
container
navbar-brand (nombre del tema)
navbar-toggler
collapse
3 links funcionales (que naveguen a secciones reales)
Debe funcionar en móvil.

2️⃣ Hero Section
Debe contener:
Título (display-*)
Texto descriptivo
Botón principal (btn btn-primary)
Espaciado (py-5)
Fondo diferenciado (bg-light, bg-dark, etc.)
El botón debe navegar a una sección real.

3️⃣ Sección de 3 Cards
Debe incluir:
row
col-12 col-md-6 col-lg-4
Imagen (img-fluid)
Título
Texto breve (máximo 3 líneas)
Botón funcional
h-100
shadow-sm

Debe verse correctamente:
📱 Móvil → 1 card
📱 Tablet → 2 cards
💻 Escritorio → 3 cards

4️⃣ Footer

Debe incluir:
Fondo oscuro
Texto centrado
Espaciado adecuado

📐 Restricciones
No usar CSS avanzado personalizado.
No usar href="#" sin funcionalidad.
No copiar código de internet.
No usar frameworks adicionales.
